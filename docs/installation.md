## CreatureCapture - Installation

Make sure you have [downloaded](download.md) the plugin first.

### Install steps

1. Stop your server.
2. Put the CreatureCapture JAR into your `plugins/` folder.
3. Start the server.
4. Confirm startup in console logs.

Expected log example:

```text
[CreatureCapture] Enabling CreatureCapture vX.X.X
```

### First checks

- Confirm no startup errors are shown.
- Verify `plugins/CreatureCapture/` was generated.
- Check that `config.yml` exists inside the plugin folder.
- Join the server and run `/cc collection` or `/cc stats`.

If you encounter issues, check [troubleshooting](troubleshooting.md).
